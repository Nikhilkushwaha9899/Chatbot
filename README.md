ğŸ’¬ How It Works
Crisis Detection System
The chatbot uses a three-tier keyword detection system:

Immediate Crisis Keywords: Triggers immediate crisis response with emergency resources

Examples: "kill myself", "end my life", "suicide", "want to die"


High-Risk Keywords: Activates crisis intervention protocols

Examples: "suicidal", "self harm", "hurt myself", "no hope"


Moderate Keywords: Provides empathetic support and resources

Examples: "depressed", "hopeless", "worthless", "alone"



Response Categories

Crisis Responses: Immediate crisis resources and safety checks
Depression Support: Validation and encouragement for those feeling hopeless
Loneliness Support: Connection strategies and empathy
Self-Harm Intervention: Professional referrals and harm reduction
Coping Strategies: Practical techniques for managing emotions
Resource Information: Crisis lines, therapy options, and support groups
